This project is sponsored by MSR Global Health which produces chlorine generators or water filteration devices for developing countries.
The main goal was to create web-based dashboards for both donors and administrative users, utilizing the data collected from the devices
As the designer of four-person group, I created wireframes, designed laytout of the dashboards, and implemented with CSS and HTML.
